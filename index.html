What We Offer

Everything you need to explore, preserve, and build vintage computing hardware and software.

Hardware Replicas

Accurate recreations of classic computers and peripherals using modern manufacturing techniques.

ROM Adapters

Bridge adapters allowing modern EEPROM chips to work with vintage systems.

KiCAD Libraries

Comprehensive component libraries for vintage chip footprints and symbols.

Documentation
